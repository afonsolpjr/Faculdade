# Quando usamos o conceito de inner class do Java?
Usamos o conceito de Inner class para criar e conceber classes que só fazem sentido dentro de uma outra classe. Permitindo modularizar ainda mais o código.

# O que é encapsulamento e qual a sua importância para POO?
Encapsulamento é a prática de proteger variaveis do acesso de outras partes do código. É importante para evitar o acesso direto de variáveis sensíveis e cruciais à outras partes do código que possam alterá-la.

# Qual a diferença entre o modificador de visibilidade private e protected?

Variáveis e métodos com o modificador private só podem ser acessadas pela própria classe.
Já com o modificador protected, classes pertencentes ao mesmo pacote e classes-filhas também terão acesso às variáveis.

# Qual é o ciclo de um desenvolvimento orientado a testes (TDD)?
Primeiro deve-se conceber testes para uma funcionalidade específica.
Depois, implementá-la de modo que passe no teste, para assim aprová-la ou testá-la novamente com novos testes.
